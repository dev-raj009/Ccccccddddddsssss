API Endpoints
GET /API/batches
Returns full list of all available batches with ID, name, price, and source platform.
GET /API/batch/:id
Returns full batch detail — total videos, subjects list, and all video links grouped by subject.
GET /API/subjects/:id
Returns subject-wise video count for a given batch ID. Quick overview without full video list.
GET /API/search?q=
Search videos across all batches by title/subject. Optional batch_id param to narrow to one batch.
GET /API/stats
Returns global stats — total batches, total videos, and breakdown by platform.
